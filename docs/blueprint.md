# **App Name**: WorldForge Chronicles

## Core Features:

- World Creation: Users can create new worlds with unique names, era, and initial race count using localStorage
- Race Naming: Generates a naming profile for a new race. Consists of phonemes, inspiration and language structure.
- Time Advancement: Advance time in increments of 1 or 10 years, with narrative events, population changes, character lifecycle updates, problem simulation, societal evolution, geographical discoveries based on Gemini Pro.
- Chronicle Entry Intervention: Enter chronicle entries as a player to influence the AI narrative simulation.
- Boon Activation: Activate 'boons' or blessings for your race that will affect their simulation cycle and may help overcome obstacles
- Cataclysm Events: Handle cataclysm events (flood, volcano, blight) with varying outcomes depending on the preparations
- Notable Character Simulation: Reasoning LLM to determine deaths for fallen notable characters and generate commoner deaths for an emotional impact using Gemini Pro tool.

## Style Guidelines:

- Primary color: Bright yellow-gold (#FFDA63) to convey a sense of divinity and fantasy.
- Background color: Dark zinc-950 (#18181B) to create an immersive dark fantasy aesthetic.
- Accent color: Yellow-orange (#FFB347) for interactive elements and important notifications.
- Headline font: 'Cinzel', a serif font, to give headlines an RPG feel. Note: currently only Google Fonts are supported.
- Body font: 'Inter', a sans-serif font, for a clean, readable style for descriptions and long-form content. Note: currently only Google Fonts are supported.
- Use consistent, detailed icons from 'lucide-react' to represent different races, boons, events, and locations. Emphasize a blend of medieval and fantasy aesthetics.
- A header banner styled as in Nemathor, containing links, user info, and a small image. Use well-placed Shadcn components to give a professional appeal.
